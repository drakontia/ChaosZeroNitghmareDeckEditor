# ChaosZeroNitghmareDeckEditor

ゲーム「カオスゼロナイトメア」のカードデッキを作成・編集できる攻略補助サイトです。

## 機能

- **キャラクター選択**: 様々なレアリティのキャラクターから選択
- **装備選択**: 武器、防具、ペンダントを装備
- **カード選択**: 複数のカードタイプに対応
  - 通常カード
  - 共用カード
  - モンスターカード
  - 禁忌カード
- **ヒラメキ機能**: カードのコストと効果を変更
  - 削除（通常状態）
  - ヒラメキ
  - 神ヒラメキ

## 技術スタック

- **Next.js 16**: 最新版のReactフレームワーク
- **TypeScript**: 型安全な開発
- **Tailwind CSS**: スタイリング
- **Playwright**: E2Eテスト

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# 本番ビルド
pnpm build

# E2Eテストの実行
pnpm test
```

## 使い方

1. **キャラクター選択**: 使用したいキャラクターをクリック
2. **装備選択**: 武器、防具、ペンダントを選択
3. **カード追加**: カードをクリックしてデッキに追加
4. **ヒラメキ設定**: デッキ内のカードの状態を変更して効果を最適化
5. **デッキ管理**: 不要なカードを削除、またはデッキ全体をクリア
